id: rectorsize
title: RectOrSize Object Type
---

`RectOrSize`接受数字像素值来描述如何扩展矩形区域。这些值会被添加到原始区域的大小上以扩展它。

## 示例

```js
{
    bottom: 20,
    left: null,
    right: undefined,
    top: 50
}
```

## Keys and values

### `bottom`

| 类型                        | Required |
| --------------------------- | -------- |
| number, `null`, `undefined` | No       |

### `left`

| 类型                        | Required |
| --------------------------- | -------- |
| number, `null`, `undefined` | No       |

### `right`

| 类型                        | Required |
| --------------------------- | -------- |
| number, `null`, `undefined` | No       |

### `top`

| 类型                        | Required |
| --------------------------- | -------- |
| number, `null`, `undefined` | No       |
